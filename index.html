Diccionario de Domótica

Domótica

Automatización de tareas en hogares mediante la integración de tecnología electrónica.

Internet de las Cosas (IoT)

Conexión de dispositivos físicos a internet para intercambiar datos y realizar acciones.

Sensor de movimiento

Dispositivo que detecta presencia humana y activa sistemas como luces automáticas o alarmas.

Termostato inteligente

Dispositivo que regula la temperatura del hogar automáticamente, optimizando el consumo energético.

Asistente de voz

Software que permite controlar dispositivos y realizar tareas mediante comandos de voz.

Iluminación inteligente

Sistema de luces que se ajustan automáticamente según la presencia, la hora del día o preferencias.

Cámara de seguridad conectada

Dispositivo que transmite imágenes y videos en tiempo real a través de internet para monitorear el hogar.

Red de malla (Mesh network)

Sistema de conexión de dispositivos donde cada nodo se comunica directamente con otros, mejorando la cobertura y estabilidad.

Automatización del hogar

Integración de tecnologías para controlar y gestionar sistemas como iluminación, seguridad y climatización de manera automática.

Control remoto

Capacidad de operar dispositivos desde una ubicación distante a través de tecnologías como internet o radiofrecuencia.

Sistema de gestión energética

Tecnología que monitorea y optimiza el consumo energético del hogar, reduciendo costos y aumentando eficiencia.

Control de persianas automatizado

Automatización del manejo de persianas o cortinas según la luz solar o las preferencias del usuario.

Interruptor inteligente

Dispositivo que permite controlar luces y electrodomésticos de forma remota mediante aplicaciones móviles.

Detector de humo conectado

Dispositivo que detecta humo y envía notificaciones inmediatas a dispositivos conectados para una respuesta rápida ante emergencias.
